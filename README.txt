Fonctionnalités réalisées :
Affichage d'une photo individuelle :
- L'utilisateur peut consulter une photo en précisant l'identifiant de la photo dans l'url
- L'utilisateur peut voir les détails d'une photo : image titre, type et catégorie
- L'utilisateur peut lire les commentaires associés à la photo

Affichage d'une galerie de photos :
- L'utilisateur peut charger une galerie de photos sous forme de vignettes
- L'utilisateur peut naviguer entre les pages de la galerie (suivante / précédente / première / dernière)
- L'utilisateur peut cliquer sur une vignette pour afficher les détails d'une photo en bas de page
